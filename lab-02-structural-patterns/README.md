Система управления организационной структурой предприятия

## Описание проекта

Информационная система для управления организационной структурой предприятия с использованием **паттерна проектирования Composite (Компоновщик)**.

Система позволяет:

- Создавать иерархическую структуру подразделений любой вложенности
- Управлять штатным расписанием (добавлять/удалять должности)
- Управлять подразделениями (добавлять/удалять отделы)
- Рассчитывать агрегированные показатели (штатные единицы, фонд оплаты труда)
- Выводить структурированное представление организации

---

## Цель работы

Изучить и применить на практике **структурный паттерн проектирования Composite** для представления древовидных иерархических структур и обеспечения единообразной работы с отдельными объектами и их группами.
