# Система комплектации автомобилей Audi A4

## Описание проекта

Информационная система для "сборки" различных комплектаций автомобиля **Audi A4** с использованием **паттерна
проектирования Builder (Строитель)**.

Система позволяет:

- Собирать различные комплектации (Базовая, S line, Premium Plus)
- Создавать кастомные конфигурации
- Генерировать HTML-отчет со сравнительной таблицей опций

---

## Цель работы

Изучить и применить на практике **порождающий паттерн проектирования Builder** для создания сложных объектов с
множеством опциональных параметров.

---

## Структура проекта

```
lab-01-creational-patterns/
│
├── Паттерн Builder - Основные компоненты
│   ├── Car.java                      # Продукт (Product)
│   ├── CarBuilder.java               # Интерфейс строителя (Builder)
│   ├── CarDirector.java              # Директор (Director)
│   │
│   └── Конкретные строители (Concrete Builders)
│       ├── AudiA4BaseBuilder.java    # Строитель базовой комплектации
│       ├── AudiA4SlineBuilder.java   # Строитель S line
│       └── AudiA4PremiumBuilder.java # Строитель Premium Plus
│
├── Утилиты
│   ├── HTMLReportGenerator.java      # Генератор HTML-отчетов
│   └── Main.java                     # Главный класс (точка входа)
│
│
└── Примеры
    └── audi_comparison.html  # Пример HTML-отчета
```
